# Python Demo Application

This is a Demo Python REST API application

## Libraries

* Environment: Pipenv
* REST Framework: Flask, Flask-RESTx
* Serialization: Marshmallow, Flask-Marshmallow
* ORM: SQLAlchemy
* DB Migration: Flask-Migrate
